# Aide-mémoire — Samuel Coaching

## Accéder à mon site en local (pour travailler)

1. Ouvre un terminal PowerShell
2. Tape ces commandes :

```
cd C:\Users\Surface\samuel-coaching
npm run dev
```

3. Ouvre ton navigateur et va sur : http://localhost:3000
4. Le terminal doit rester ouvert pendant que tu travailles

---

## Mettre mon site en ligne (après avoir fait des modifications)

Une fois tes modifications faites, tape ces 3 commandes dans le terminal :

```
git add .
git commit -m "décris ce que tu as changé"
git push
```

Attends 1-2 minutes puis ton site sur Vercel est mis à jour automatiquement.
